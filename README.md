
# Telegram Store: Зачпасти у Алёшки

Это готовый пакет Telegram-магазина автозапчастей с базой товаров, подготовленной из вашего прайса.
Файлы внутри:
- bot.py — основной бот (aiogram)
- catalog_full.sqlite — база с товарами (поле parts)
- catalog_full.xlsx — исходный подготовленный прайс (полезно для правок)
- Прайс оригинал: /mnt/data/Прайс 21.11.xlsx
- images/ — папка для изображений (если хотите, можно загрузить сюда файлы с именами OEM.jpg)
- requirements.txt — зависимости

## Быстрый запуск (локально)

1. Установите Python 3.9+ и зависимости:
```bash
pip install -r requirements.txt
```

2. Установите переменную окружения с токеном бота и (необязательно) путь к БД:
```bash
export BOT_TOKEN='ВАШ_ТОКЕН_ЗДЕСЬ'
export CATALOG_DB='catalog_full.sqlite'
```

3. Запустите бота:
```bash
python bot.py
```

## Примечания
- Для поиска изображений используйте скрипт image_finder.py (не включён в пакет). В images/ кладём файлы с именами OEM.jpg — бот подставит их автоматически.
- В каталоге уже есть `catalog_full.sqlite` и `catalog_full.xlsx` (подготовленные ранее).
- Путь к оригинальному файлу прайса: `/mnt/data/Прайс 21.11.xlsx` (см. файл).

Название магазина: **Зачпасти у Алёшки**
